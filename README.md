# Grove for Arduino Uno

Port declaration for Grove Shield V2.0

- GROVE_A0 : (A0,A1)
- GROVE_A1 : (A1,A2)
- GROVE_A2 : (A2,A3)
- GROVE_A3 : (A3,A4)
- GROVE_D2 : (2,3)
- GROVE_D3 : (3,4)
- GROVE_D4 : (4,5)
- GROVE_D5 : (5,6)
- GROVE_D6 : (6,7)
- GROVE_D7 : (7,8)
- GROVE_D8 : (8,9)
- GROVE_I2C : (A4,A5)
- GROVE_UART : (0,1)
